# AI Agent: Health, Translation, Emergency, and Mental Health Companion

An AI agent built with Streamlit that provides the following features:
1. **Health and Wellness Coach**: Tracks steps and water intake, suggests workouts, and provides meal ideas.
2. **Language Translator**: Translates text and speaks it aloud in the target language.
3. **Emergency Helper**: Provides first-aid steps and locates nearby help during emergencies.
4. **Mental Health Companion**: Offers breathing exercises, mood check-ins, and daily tips.

## Features
- **Health and Wellness Coach**:
  - Daily step and water intake tracking.
  - Suggests simple home workouts or stretches.
  - Provides healthy meal ideas based on dietary preferences.
- **Language Translator**:
  - Real-time translation for spoken or written text.
  - Teaches common phrases for travelers.
  - Speaks translated text aloud.
- **Emergency Helper**:
  - Guides users through first-aid steps.
  - Locates nearby hospitals, pharmacies, or shelters.
- **Mental Health Companion**:
  - Guided breathing exercises or meditation.
  - Daily mood check-ins.
  - Daily mental health tips.
